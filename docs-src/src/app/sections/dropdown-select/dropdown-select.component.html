Dropdown Select

Installation
  
  // install the repo
  npm install CINBCUniversal/suprematism-dropdown-select --save

  // add it to a module (app.module.ts)
  import {{'{'}} DropdownSelectModule } from 'suprematism-dropdown-select';

  @NgModule({{'{'}}
    ...
    imports: [
      ...
      DropdownSelectModule
    ],
    ...
  })
Simple Example
Selection : {{heroes.join(',')}} 
  
  {{'<'}}supre-dropdown-select
    [value]="[]"
    allLabel="All Star Wars Heroes"
    prompt="Select Start Wars Heroes"
    [options]="availableHeroes"
    (valueChange)="heroes=$event.value">
  {{'<'}}/supre-dropdown-select>
Treatment of Empty Selected Values
Sometimes you may want to treat an empty value as "All". Setting `emptyValueMode="all"` will show all selected if `value` is empty and set value to an empty array when all options are selected. If no options are selected `value` will contain a single value determined by the `noneSelectedValue` property which defaults to -1.
Villains: {{villains.join(',')}} 
  
    {{'<'}}supre-dropdown-select
      [value]="[]"
      emptyValueMode="all"
      noneSelectedValue="none"
      allLabel="All Star Wars Villains"
      prompt="Select Star Wars Villain"
      [options]="availableVillains"
      (valueChange)="villains=$event.value">
    {{'<'}}/supre-dropdown-select>
Button Label

You can specify a custom buttonLabel for the dropdown button.

Planets: {{planets.join(',')}} 
  
    {{'<'}}supre-dropdown-select
      [buttonLabel]="getPlanetsLabel(model.planets)"
      allLabel="All Planets"
      [options]="availablePlanets"
      (valueChange)="planets=$event.value"
      >
    {{'<'}}/supre-dropdown-select>
Trigger Events

Set an openTrigger and closeTrigger to control if the dropdown opens on hover or click. The default openTrigger is 'click' and closeTrigger is 'hover'.

Open=Click, Close=Hover (default)
Open=Click, Close=Click
Open=Hover, Close=Hover
Open=Hover, Close=Click
  
    {{'<'}}supre-dropdown-select
      prompt="Select Droid"
      openTrigger="hover"
      closeTrigger="click"
      allLabel="All Droids"
      [options]="availableDroids">
    {{'<'}}/supre-dropdown-select>
Hiding the "All" option

You can hide the "All" option by setting showAllOption=false

  
    {{'<'}}supre-dropdown-select
      emptyValue="none"
      [showAllOption]="false"
      prompt="Select Start Wars Villain"
      [options]="availableVillains">
    {{'<'}}/supre-dropdown-select>
