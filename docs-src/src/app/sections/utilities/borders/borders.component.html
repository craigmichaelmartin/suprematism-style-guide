Border Placeholders

This will add border styles to any class you use it on.
Border placehoders are a Utility, add it to your SCSS file from here:

  
  @import "../node_modules/suprematism-style-guide/dist/utils/_borders.scss";
  

  
  // use the @extend directive, placholders start with '%', not a '.'
  
  @extend %_b0;  // border: none !important;
  @extend %_bt0; // border-top: none !important;
  @extend %_br0; // border-right: none !important;
  @extend %_bb0; // border-bottom: none !important;
  @extend %_bl0; // border-left: none !important;
  
  @extend %_b1;  // border: 1px solid {{'<'}}color: $active-stroke>;
  @extend %_bt1; // border-top: 1px solid {{'<'}}color: $active-stroke>;
  @extend %_br1; // border-right: 1px solid {{'<'}}color: $active-stroke>;
  @extend %_bb1; // border-bottom: 1px solid {{'<'}}color: $active-stroke>;
